## 🎭 IMDB Sentiment Analysis using NLP

A Natural Language Processing (NLP) project built using Python and Jupyter Notebook that classifies IMDB movie reviews as positive or negative based on textual content. The project demonstrates text preprocessing, feature extraction, sentiment classification, model evaluation, and predictive analytics using a real-world movie review dataset.

## 🚀 Features
* **Sentiment Analysis System** – Classify movie reviews as positive or negative.
* **Natural Language Processing Pipeline** – Clean and preprocess textual data for machine learning.
* **Text Vectorization** – Convert reviews into numerical feature representations.
* **Binary Text Classification** – Predict sentiment from unseen movie reviews.
* **Feature Engineering for NLP** – Extract meaningful textual features for model training.
* **Data Cleaning & Preprocessing** – Remove stopwords, punctuation, and normalize text.
* **Model Training & Evaluation** – Train and evaluate sentiment classification models.
* **Performance Metrics Analysis** – Measure accuracy, precision, recall, and F1-score.
* **Review Analytics** – Analyze sentiment distribution across movie reviews.
* **Real-World IMDB Dataset** – Practice sentiment analysis using thousands of labeled reviews.

## 🛠️ How to Run
1. **Clone the repository**:
```
git clone https://github.com/Agent-A345/IMDB-Sentiment-Analysis-NLP.git
```
2. **Navigate to the project folder**:
```
cd IMDB-Sentiment-Analysis-NLP
```
3. **Install required libraries**
```
pip install pandas numpy matplotlib seaborn jupyter scikit-learn nltk
```
4. **Run the notebook**
```
jupyter notebook
```
5. **Open**
```
Sentiment_Analysis_NLP_Project.ipynb
```
